# REBORN - Accessibility-Focused Mobile Application

## Overview

REBORN is a mobile application designed to improve accessibility and independence for people with disabilities. The app offers tools and features that help users navigate public spaces, interact with their environment, and receive real-time assistance, aiming to enhance everyday mobility and safety.

## Features

* **Accessible Location Map:** Provides real-time information on wheelchair-accessible locations in my hometown (Buzau) including public institutions, stores, restaurants, and entertainment venues.
* **Visual Assistant:** AI-powered camera-based guidance for visually impaired users to identify objects.
* **Notification System:** Alerts users of predefined sounds (doorbells, alarms, sirens) via vibrations and flash notifications.
